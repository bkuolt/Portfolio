============================================
Features
============================================
- fixed function pipeline
- multi texturing (using lightmaps)
- collision detection
- view frustum culling
- (Quake2) BSPS

============================================
Controls
============================================

Keyboard
--------------
ESC   | exit
WASD  | move around

Mouse 
------------------------------------
left/right     | rotate left/right
scroll up/down | hower up/down
left mouse key | shoot
