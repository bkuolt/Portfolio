============================================
Features
============================================
- C++
- fixed function pipeline
- projective shadows
- infinite directional light 
- smooth / flat shading
- fog
- vbo
- mesh (Wavefront OBJ)

============================================
Controls
============================================
ESC | exit
T   | texturing on/off
L   | lighting on/off
S   | toggle smooth/flat Shading
+/- | rotate light source
